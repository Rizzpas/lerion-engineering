## 🏗️ About The Project

This is the official front-end web platform for **Lieron Engineering Limited**. It is designed to showcase high-stakes infrastructure capabilities, structural steel detailing, and a robust portfolio of New Zealand-based projects. 

The UI is built with a heavy emphasis on brutalist, modern engineering aesthetics—utilizing a high-contrast grayscale palette with signature safety-orange accents, bold typography, and strict grid alignments.

### ✨ Key Features
* **Fully Responsive:** Seamlessly scales from mobile devices to ultra-wide desktop monitors.
* **Dynamic Portfolio Grid:** Custom CSS Grid/Masonry layout for the project archive.
* **Interactive Locations:** Integrated Google Maps embed for global HQ and retail branch tracking.
* **Lead-Gen Inquiry Flow:** Custom "Start Your Build" consultation form with tailored project type selection.
* **Scroll Animations:** Smooth reveal animations powered by AOS (Animate On Scroll).
